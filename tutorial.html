- Movement

  • Use the following keys to move around the board:
  • W = move forward
  • A = move left
  • S = move down
  • D = move right

- Camera

  • Use the following arrow keys to move around the grid:
  • UP = move grid up
  • LEFT = move grid left
  • DOWN = move grid down
  • RIGHT = move grid right

- Pushable blocks

  • Use the movement keys and move into a pushable block to push the block in the direction of movement

- Target spaces

  • Target spaces should be filled with pushable blocks to advance to the next level

- Pullable blocks

  • Pull these blocks using SHIFT + movement keys the block will be pulled in the direction of movement
  • Target spaces can also be filled using these blocks

- Pullable blocks (cont.)

  • Pullable blocks occupy the same target spaces with pushable blocks
  • Either one can occupy a target space

- Borders

  • Target spaces can have borders around them which prevent blocks from entering in that direction

- Teleporters

  • These blocks can teleport pushable and pullable blocks from one space to another (including the player)

- ID blocks

  • IDs can occur in blocks and target spaces
  • Blocks with matching IDs should be moved to target spaces with matching IDs
  • Blocks and targets with mismatching IDs will light up as a different color

- Import & Export

  • Use the following key combinations to import or export a file:
  • CMD+I/CTRL+I = import a level file to load
  • CMD+E/CTRL+E = export level data into a JSON file
